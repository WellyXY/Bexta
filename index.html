P1
Add a "Share link" shortcut before the deploy step
82% of users never reached deploy. A simple "copy link to share" option would let users complete their goal without needing domain knowledge.
↑ Est. +18% conversion
P2
Show example prompts on the blank input screen
34% of users stalled 30+ seconds at the prompt screen. Add 3 tappable example prompts ("Build me a landing page for...") to reduce blank-page paralysis.
↑ Est. +11% at prompt step
P3
Show a persistent build progress indicator
28 clients thought the app had crashed during build. A visible progress bar or animated status message reduces abandonment while waiting.
↑ Est. +8% retention at build step
P4
Rewrite the landing page headline for non-technical users
22% dropped before signup. Mobile users (mostly 18–25) didn't understand "vibe-code your app" — plain language like "build a real website by describing it" tested better.
↑ Est. +7% signup rate
P5
Optimize edit panel for mobile — current overlay is too small
Mobile users (Klaus, Priya, Léa) all struggled with the edit panel tap targets. Recommend bottom-sheet pattern on screens under 430px.
